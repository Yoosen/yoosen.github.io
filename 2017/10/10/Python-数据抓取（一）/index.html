控制结构

if __name__ == ‘__main__‘:

第一个程序中“if __name__ == ‘__main__‘:”语句后正常执行，当其他程序import第一个程序中的函数时，不会执行第一个程序“if __name__ == ‘__main__‘:”后面的语句，避免了其他程序重复执行第一个程序的所有代码，实现按需取需要的部分。

注意：
